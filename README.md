# 🎨 Pixel Facturas

> Aplicación de escritorio para generar facturas profesionales en USD.

[![Windows](https://img.shields.io/badge/Windows-10%2F11-blue?style=flat&logo=windows)](https://github.com/luisalgomez111/Facturas/releases)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![GitHub release](https://img.shields.io/github/v/release/luisalgomez111/Facturas?style=flat)](https://github.com/luisalgomez111/Facturas/releases)

---

## ✨ Características

| Módulo | Descripción |
|--------|------------|
| 📊 **Dashboard** | Estadísticas de ingresos, facturas pendientes, historial |
| 📄 **Nueva Factura** | Crear facturas con cliente y servicios |
| 📦 **Catálogo** | Gestión de servicios y precios |
| 👥 **Clientes** | Directorio de clientes frecuentes |
| ⚙️ **Perfil** | Datos, logo, métodos de pago |
| 📑 **PDF** | Generación de facturas en PDF |

### Métodos de Pago Soportados
- 🟡 **Binance** (USDT)
- 🔵 **Transferencia** bancaria
- 📱 **Pago Móvil** (con QR)

---

## 📦 Instalación (Para Usuarios)

1. Descarga el [.zip del releases](https://github.com/luisalgomez111/Facturas/releases)
2. Extrae la carpeta
3. Ejecuta `Pixel Facturas.exe`

¡Listo! No necesitas instalar nada.

---

## 💻 Desarrollo (Para Programadores)

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Build producción
npm run build
npm run dist
```
